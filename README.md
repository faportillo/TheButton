# the-button

Blank Python project using Poetry for dependency management and packaging.

## Quickstart

1. Ensure Python 3.13 is installed and on your PATH.
2. Install Poetry: https://python-poetry.org/docs/#installation
3. Configure Poetry to use an in-project virtualenv (optional):
   ```bash
   poetry config virtualenvs.in-project true
   ```
4. Use Python 3.13 and install:
   ```bash
   poetry env use 3.13
   poetry install
   ```
5. Run the placeholder tests:
   ```bash
   poetry run pytest
   ```
